# Demo 01 Evaluations

Standardized evaluation runs for `demo01` should be stored under:

```text
output/model_evaluations/demo01/<run_id>/
```

Each run must include:

- `evaluation_manifest.json`
- `metrics.json`
- `evaluation_summary.md`

Demo 01 is frozen as a demo model. New evaluations should primarily be used as a baseline for comparison, not as a signal to keep optimizing this architecture.
